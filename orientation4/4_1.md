(W) Using an example case (health centre, bank, etc.), discuss the meaning and characterisation of all the performance measures discussed in the lesson material using the concepts of the target system.


- A Arrived Customers Count
-  The total number of customers who enter the queue

- B Busy time
-  The total time the teller spends serving customers

- C Completed customers count
-  The total number of customers who are fully served

- N Average number of customers in the system
-  Average queue length invluding the customer being served
-  N = W / T

- R Average response time
-  Average total time spent in the system
-  R = W / C

- S Average service time
-  Average time spent actually serving a customer
-  S = B / C

- T total simulation time
-  The total observed time

- U utilization
-  The proportion of time the teller is busy
-  U = B / T

- W Total response time
-  Sum of all customers' response times

- X Throughput
-  The number of customers served per unit of time
-  X = C / T

- RI Individual response time
-  The time a single customer spends in the system
