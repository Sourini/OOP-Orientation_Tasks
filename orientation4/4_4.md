(W) Write an algorithm in Java or pseudocode for the main loop of the simulator:

The main loop does all the things required for a three-phase simulation (presented in a section).

Think in general terms (don't think in detail) and present a general solution. No need for target system concepts.


pseudocode main simulation loop:

```java
initializeSimulation();

while (!simulationFinished()) {

    // A-phase
    Event nextEvent = eventList.removeNextEvent();
    clock.setTime(nextEvent.getEventTime());

    // B-phase
    processEvent(nextEvent);

    while (!eventList.isEmpty() &&
           eventList.peekNextEventTime() == clock.getTime()) {
        processEvent(eventList.removeNextEvent());
    }

    // C-phase
    boolean eventExecuted;
    do {
        eventExecuted = false;

        for (ServicePoint sp : servicePoints) {
            if (sp.canStartService()) {
                sp.startService();
                eventExecuted = true;
            }
        }
    } while (eventExecuted);
}

printResults();
```
